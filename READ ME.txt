This project is based on Topic-COUNTING(A Wide field of Discrete Mathematics).

In this we can calculate npr,ncr,binomial expansion,value of binomial expansion,arrangements of strings.



1. Algorithm Used:
Recursion,BinaryExponentiation.


2.Functions used:

*fact(n):-  It calculates the Factorial of n.

*npr(num1,num2):-It calculates npr,where n=num1,r=num2.

* ncr(num1,num2):- It calculates ncr,where n=num1,r=num2.

*ex():- This function expands (a+b)^n and calculate the value.

*exponent(a1,b1,n1):- This function calculate (a1+b1)^n

*permute(word):-It will take the word and it will find all the arrangements of the word.
